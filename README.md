# Quick Capture

Built for personal use. Press **⌘⇧Space** anywhere → type → **⌘↵** → file lands in knowledge base inbox. 

## Build & install

```bash
./build-app.sh
./build-app.sh install
open /Applications/QuickCapture.app
```

## What lands in the inbox

```yaml
---
captured: 2026-05-22T03:24:16Z
hotkey: cmd-shift-space
app: "Safari"
window: "Some article title"
url: "https://example.com/article"
---

<the body you typed>
```

## Menu bar

Click the pencil icon → `Capture…`, `Launch at Login` (toggle), `Quit Quick Capture`.
